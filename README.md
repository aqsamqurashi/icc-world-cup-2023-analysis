# ICC Cricket World Cup 2023 — Data Analysis

## About
Ball-by-ball analysis of the ICC Cricket World Cup 2023 using MySQL and Python.
The dataset contains 26,000+ deliveries across 48 matches and 10 teams.

## Questions Answered
1. How many total matches were played?
2. Which teams participated and how many times did each team bat first?
3. Which venues hosted the most matches?
4. Who are the top 10 run scorers?
5. Which batsmen hit the most boundaries?
6. Top 3 batsmen from each team?
7. Which bowlers took the most wickets?
8. Which bowlers had the best economy rate?
9. Which bowlers never bowled a no ball?
10. Total runs and winner for each match?
11. How many matches did each team win?
12. Batting first vs bowling first wins per venue?
13. Overall tournament batting rankings
14. Top batsman ranking per match
15. Running total of runs ball by ball

## Key Findings
- V Kohli was the top scorer with 765 runs
- India went unbeaten in the group stage
- Australia won the final against India
- Batting first and bowling first wins were roughly equal across venues

## Tools Used
- MySQL — data storage and querying
- Python — pandas, SQLAlchemy
- Jupyter Notebook — analysis and presentation
